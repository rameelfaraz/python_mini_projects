# Student Grade Analyzer

A Python mini project that reads student data from a CSV file, calculates academic statistics, assigns letter grades, and saves the final report to a JSON file.

## Features
- Load student names and marks from a CSV file
- Validate marks and skip invalid entries
- Calculate average marks
- Find highest and lowest marks
- Assign letter grades to each student
- Display student results through a menu-driven interface
- Save the final report to a JSON file

## How to Run
1. Open terminal and navigate to the project folder
2. Run: python student_grade_analyzer.py

## Required CSV Format

The students.csv file should contain data in this format:

name,marks
Ali Khan,85
Sara Ahmed,92
Ahmed Raza,58
Hina Fatima,76

## Output in JSON

The porgam stores the report in a file named "student_report.json"
