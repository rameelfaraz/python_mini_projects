# Password Tool

A Python mini project that can **generate secure passwords** and **check password strength** with helpful feedback.

## Features
- Generate random passwords of any valid length
- Includes letters, numbers, and symbols
- Ensures stronger generated passwords for lengths 3 and above
- Check password strength (Weak, Medium, Strong)
- Gives feedback to improve weak passwords
- Input validation for password length and menu choices

## How to Run

1. Clone the repository:
   git clone <repo-link>

2. Navigate to the project folder:
   cd python_mini_projects/password_generator

3. Run the program: 
   python password_generator.py
