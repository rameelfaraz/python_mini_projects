# Wes Anderson Quiz Game

A Python text-based quiz game themed around **Wes Anderson movies**.

## Features
- Three difficulty levels: Easy, Medium, Hard
- Interactive character dialogue (Monsieur Gustave)
- Fuzzy matching for slightly incorrect answers
- Tracks score and gives feedback based on performance
- Replayable without restarting the program

## How to Run

1. Clone the repository:
   git clone <repo-link>

2. Navigate to the project folder:
   cd python_mini_projects/wes_anderson_quiz

3. Run the program: 
   python wes_anderson_quiz.py
